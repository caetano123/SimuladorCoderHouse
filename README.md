Simulador de Ahorro Mensual

Autor: Caetano Elizalde

📝 ¿Qué es este simulador?

Este proyecto es una herramienta simple desarrollada en JavaScript que permite simular cuánto dinero se puede ahorrar mes a mes. El usuario ingresa un monto fijo mensual y la cantidad de meses que desea ahorrar, y el simulador le devuelve el total acumulado al finalizar ese período. Además, se muestra un detalle del ahorro acumulado mes por mes en la consola del navegador.

🎯 Objetivos personales del proyecto

- Practicar el uso de variables, constantes y arrays en JavaScript.
- Crear funciones organizadas con entradas, procesamiento y salida de datos.
- Aplicar condicionales y bucles (como `if` y `for`) de forma lógica y útil.
- Mejorar mi capacidad para interactuar con el usuario usando `prompt`, `confirm` y `alert`.
- Mostrar información clara y útil en la consola, incluyendo un resumen y un desglose mensual.
- Sentar una base de lógica que luego pueda expandirse con más funcionalidades (por ejemplo, guardar datos o agregar una interfaz visual).

💻 Tecnologías utilizadas

- HTML5 (estructura básica del proyecto)
- JavaScript (toda la lógica del simulador)
- Consola del navegador (para mostrar resultados)
- Cuadros de diálogo (`prompt`, `confirm`, `alert`) para interactuar con el usuario

📂 Organización del proyecto

- index.html → Contiene la descripción general del simulador y el enlace al archivo JS.
- js/simulador.js → Contiene todo el código JavaScript del simulador.
